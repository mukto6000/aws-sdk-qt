Detailed Description

The ImportExportRequest class provides an interface for ImportExport requests.

Member Type Documentation

enum ImportExportRequest::Action
flags ImportExportRequest::Actions

This enum describes the actions that can be performed as ImportExport requests.

ConstantValueDescription
QtAws::ImportExport::ImportExportRequest::CancelJobAction0ImportExport CancelJob action.
QtAws::ImportExport::ImportExportRequest::CreateJobAction1ImportExport CreateJob action.
QtAws::ImportExport::ImportExportRequest::GetShippingLabelAction2ImportExport GetShippingLabel action.
QtAws::ImportExport::ImportExportRequest::GetStatusAction3ImportExport GetStatus action.
QtAws::ImportExport::ImportExportRequest::ListJobsAction4ImportExport ListJobs action.
QtAws::ImportExport::ImportExportRequest::UpdateJobAction5ImportExport UpdateJob action.

The Actions type is a typedef for QFlags<Action>. It stores an OR combination of Action values.

[override virtual protected] QNetworkRequest ImportExportRequest::unsignedRequest(const QUrl &endpoint) const

Reimplemented from AwsAbstractRequest::unsignedRequest().

Returns a network request for the ImportExport request using the given endpoint.

This ImportExport implementation builds request URLs by combining the common query parameters (such as Action and Version), with any that have been added (via setParameter) by child classes.

[virtual] bool ImportExportRequest::operator==(const ImportExportRequest &other) const

Returns true if this request is equal to other; false otherwise.
