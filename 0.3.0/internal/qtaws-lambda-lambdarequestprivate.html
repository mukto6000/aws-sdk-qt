LambdaRequestPrivate::LambdaRequestPrivate(const LambdaRequest::Action action, LambdaRequest * const q)

Constructs a LambdaRequestPrivate object for Lambda action, with public implementation q.

LambdaRequestPrivate::LambdaRequestPrivate(const LambdaRequestPrivate &other, LambdaRequest * const q)

Constructs a copy of other, with public implementation q.

This copy-like constructor copies everything from other, except for the the object's pointer to its public instance - for that, q is used instead.

This is required to support the LambdaRequest class's copy constructor.

[static] QString LambdaRequestPrivate::toString(const LambdaRequest::Action &action)

Returns a string represention of action, or a null string if action is invalid.

This function converts LambdaRequest::Action enumerator values to their respective string representations, appropriate for use with the Lambda service's Action query parameters.
