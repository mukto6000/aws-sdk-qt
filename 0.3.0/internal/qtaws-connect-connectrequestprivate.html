ConnectRequestPrivate::ConnectRequestPrivate(const ConnectRequest::Action action, ConnectRequest * const q)

Constructs a ConnectRequestPrivate object for Connect action, with public implementation q.

ConnectRequestPrivate::ConnectRequestPrivate(const ConnectRequestPrivate &other, ConnectRequest * const q)

Constructs a copy of other, with public implementation q.

This copy-like constructor copies everything from other, except for the the object's pointer to its public instance - for that, q is used instead.

This is required to support the ConnectRequest class's copy constructor.

[static] QString ConnectRequestPrivate::toString(const ConnectRequest::Action &action)

Returns a string represention of action, or a null string if action is invalid.

This function converts ConnectRequest::Action enumerator values to their respective string representations, appropriate for use with the Connect service's Action query parameters.
