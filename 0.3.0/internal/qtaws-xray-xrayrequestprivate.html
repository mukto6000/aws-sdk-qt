XRayRequestPrivate::XRayRequestPrivate(const XRayRequest::Action action, XRayRequest * const q)

Constructs a XRayRequestPrivate object for XRay action, with public implementation q.

XRayRequestPrivate::XRayRequestPrivate(const XRayRequestPrivate &other, XRayRequest * const q)

Constructs a copy of other, with public implementation q.

This copy-like constructor copies everything from other, except for the the object's pointer to its public instance - for that, q is used instead.

This is required to support the XRayRequest class's copy constructor.

[static] QString XRayRequestPrivate::toString(const XRayRequest::Action &action)

Returns a string represention of action, or a null string if action is invalid.

This function converts XRayRequest::Action enumerator values to their respective string representations, appropriate for use with the XRay service's Action query parameters.
