AcmRequestPrivate::AcmRequestPrivate(const AcmRequestPrivate &other, AcmRequest * const q)

Constructs a copy of other, with public implementation q.

This copy-like constructor copies everything from other, except for the the object's pointer to its public instance - for that, q is used instead.

This is required to support the AcmRequest class's copy constructor.

[static] QString AcmRequestPrivate::toString(const AcmRequest::Action &action)

Returns a string represention of action, or a null string if action is invalid.

This function converts AcmRequest::Action enumerator values to their respective string representations, appropriate for use with the ACM service's Action query parameters.
