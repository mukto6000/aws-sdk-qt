SmsRequestPrivate::SmsRequestPrivate(const SmsRequest::Action action, SmsRequest * const q)

Constructs a SmsRequestPrivate object for SMS action, with public implementation q.

SmsRequestPrivate::SmsRequestPrivate(const SmsRequestPrivate &other, SmsRequest * const q)

Constructs a copy of other, with public implementation q.

This copy-like constructor copies everything from other, except for the the object's pointer to its public instance - for that, q is used instead.

This is required to support the SmsRequest class's copy constructor.

[static] QString SmsRequestPrivate::toString(const SmsRequest::Action &action)

Returns a string represention of action, or a null string if action is invalid.

This function converts SmsRequest::Action enumerator values to their respective string representations, appropriate for use with the SMS service's Action query parameters.
