KmsRequestPrivate::KmsRequestPrivate(const KmsRequestPrivate &other, KmsRequest * const q)

Constructs a copy of other, with public implementation q.

This copy-like constructor copies everything from other, except for the the object's pointer to its public instance - for that, q is used instead.

This is required to support the KmsRequest class's copy constructor.

Returns a string represention of action, or a null string if action is invalid.

This function converts KmsRequest::Action enumerator values to their respective string representations, appropriate for use with the KMS service's Action query parameters.
