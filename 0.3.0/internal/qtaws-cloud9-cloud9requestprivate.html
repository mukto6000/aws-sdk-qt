Cloud9RequestPrivate::Cloud9RequestPrivate(const Cloud9RequestPrivate &other, Cloud9Request * const q)

Constructs a copy of other, with public implementation q.

This copy-like constructor copies everything from other, except for the the object's pointer to its public instance - for that, q is used instead.

This is required to support the Cloud9Request class's copy constructor.

[static] QString Cloud9RequestPrivate::toString(const Cloud9Request::Action &action)

Returns a string represention of action, or a null string if action is invalid.

This function converts Cloud9Request::Action enumerator values to their respective string representations, appropriate for use with the Cloud9 service's Action query parameters.
