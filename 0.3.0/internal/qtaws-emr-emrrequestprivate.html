EmrRequestPrivate::EmrRequestPrivate(const EmrRequestPrivate &other, EmrRequest * const q)

Constructs a copy of other, with public implementation q.

This copy-like constructor copies everything from other, except for the the object's pointer to its public instance - for that, q is used instead.

This is required to support the EmrRequest class's copy constructor.

[static] QString EmrRequestPrivate::toString(const EmrRequest::Action &action)

Returns a string represention of action, or a null string if action is invalid.

This function converts EmrRequest::Action enumerator values to their respective string representations, appropriate for use with the EMR service's Action query parameters.
