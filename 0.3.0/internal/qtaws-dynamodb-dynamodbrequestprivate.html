Constructs a DynamoDBRequestPrivate object for DynamoDB action, with public implementation q.

DynamoDBRequestPrivate::DynamoDBRequestPrivate(const DynamoDBRequestPrivate &other, DynamoDBRequest * const q)

Constructs a copy of other, with public implementation q.

This copy-like constructor copies everything from other, except for the the object's pointer to its public instance - for that, q is used instead.

This is required to support the DynamoDBRequest class's copy constructor.

[static] QString DynamoDBRequestPrivate::toString(const DynamoDBRequest::Action &action)

Returns a string represention of action, or a null string if action is invalid.

This function converts DynamoDBRequest::Action enumerator values to their respective string representations, appropriate for use with the DynamoDB service's Action query parameters.
