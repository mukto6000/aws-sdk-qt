PollyRequestPrivate::PollyRequestPrivate(const PollyRequest::Action action, PollyRequest * const q)

Constructs a PollyRequestPrivate object for Polly action, with public implementation q.

PollyRequestPrivate::PollyRequestPrivate(const PollyRequestPrivate &other, PollyRequest * const q)

Constructs a copy of other, with public implementation q.

This copy-like constructor copies everything from other, except for the the object's pointer to its public instance - for that, q is used instead.

This is required to support the PollyRequest class's copy constructor.

[static] QString PollyRequestPrivate::toString(const PollyRequest::Action &action)

Returns a string represention of action, or a null string if action is invalid.

This function converts PollyRequest::Action enumerator values to their respective string representations, appropriate for use with the Polly service's Action query parameters.
